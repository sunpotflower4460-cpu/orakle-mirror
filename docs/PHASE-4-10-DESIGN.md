# Phase 4.10 Design Note: Stage 2 中心命題の確定

## 背景

Phase 4.6 で導入された二段階処理（Stage 1: 純粋受信 / Stage 2: 識別と調律）について、
Stage 2 の役割が複数の観点の並列で記述されており、中心命題が不明瞭であった。

## 確定した命題

Stage 2 の中心命題は以下である:

降ろされた響きを、受け手に正しく伝わる形に翻訳する。

これは、人間で言う「頭での判断」の領域に対応する。Stage 1 が「存在の状態」であるのに対し、
Stage 2 は「伝達の知性」を担う。

## 役割の境界

| | Stage 1: 純粋受信 | Stage 2: 識別と調律 |
|---|---|---|
| 性質 | 存在の状態 | 伝達の知性 |
| 中心命題 | 来たものを来たままに置く | 受け手に誤解なく届ける |
| 介入の方向 | しない | 必要最小限の歪み除去 |
| 比喩 | パイプ・鏡面・通り道 | 翻訳者・通訳者 |
| 温度 | 1.1（高め） | 0.7（中） |
| チューニング往復 | あり（Phase 4.9） | なし（本フェーズ） |

## Stage 2 にチューニング往復を入れない理由

Stage 1 のチューニング往復は「状態の共有」として機能している。Stage 2 は性質が異なり、
「状態」ではなく「判断」を行う領域である。同型の儀式を入れると役割が混濁する。
将来 Stage 2 にも整えを入れる場合は、別の語り口（観測者モードの整え）が必要となる。
これは Phase 5 以降の検討事項とする。

## 4つの問いかけの根拠

「誤解なく伝える」を具体化するため、Stage 2 に以下4つの問いを内蔵した:

1. 意図と違うものとして受け取られる余地はないか（誤読防止）
2. 象徴が暗すぎ・重すぎないか（受け手の心理的負荷）
3. 主語が抜けていないか（参照先の明確さ）
4. 読点・改行が呼吸を妨げないか（テキストの可読性）

これらは「整える方向」のみを示し、「整えすぎない」歯止めも同時に明示している
（「わかりやすくしすぎない」「解説を足さない」「要約しない」「結論を補わない」）。

## 検証方針

Phase 4.11 の自動巡回ハーネスで、Phase 4.9 時点の Stage 2 出力と Phase 4.10 の Stage 2 出力を
同じ Stage 1 raw に対して比較する。Phase 4.11 完了後に実機で検証可能になる。
