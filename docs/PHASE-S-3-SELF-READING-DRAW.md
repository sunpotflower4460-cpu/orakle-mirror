# Phase S-3 Self Reading Draw Ritual

Phase S-3 adds the local Self Reading draw ritual after deck/spread/question setup.

## Scope

- Ready Self Reading decks can start a local draw with `drawCards`.
- The visual ritual is symbolic: shuffle uses six dummy card backs, not the full deck.
- The draw stage deals and flips 1 / 2 / 3 cards based on the selected spread.
- Flipped cards remain visible with a small placeholder note for the detailed Phase S-4 result view.

## Non-goals

- No AI interpretation, BFF call, usage consumption, history storage, sharing, or result panel is introduced.
- AI Mirror chat and existing card reading behavior remain unchanged.
